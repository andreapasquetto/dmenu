# dmenu

A dynamic menu fox X written in C.

## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/andreapasquetto/dmenu.git
    ```

2. Compile and install

    ```sh
    make && sudo make clean install
    ```
